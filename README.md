# 🇮🇳 JanSoochna (जनसूचना)
### **Crowdsourced Civic Issue Reporting & Resolution Platform**

JanSoochna is a premium, AI-powered digital portal designed to bridge the gap between citizens and authorities. Built with a "Digital Bharat" aesthetic, it empowers users to report civic issues using voice, track resolutions in real-time, and hold departments accountable through transparent analytics.

---

## 🌟 Key Features

### 🎙️ Voice-First Experience
- **Jan Assistant**: An intelligent AI chatbot that speaks to you and listens to your queries using the Web Speech API.
- **Voice Reporting**: Tap the mic icon in the complaint form to speak your issue title and description instead of typing.
- **Narrated Details**: One-click "Read Description" feature to hear complaint details read aloud.

### 🛡️ Civic Governance
- **Official Dashboard**: A dedicated space for department officials to manage, track, and resolve assigned issues.
- **Proof of Resolution**: Officials must upload a "Proof of Work" photo to verify that an issue has been fixed.
- **Cross-Verification**: Citizens earn "Impact Points" by verified resolutions in their neighborhood.

### 📊 Transparency & Insights
- **Public Stats**: Real-time analytics on department performance and issue resolution rates.
- **AI Insights**: Automatically generated summaries and priority scores for every report.
- **Emergency SOS**: A tricolor-branded quick-access panel for immediate civic emergencies with location logging.

### 📱 Multi-Platform
- **Web Portal**: Edge-to-edge "Fluid Utility" design for desktop and mobile browsers.
- **Mobile App**: A dedicated React Native application with offline sync and real-time socket notifications.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Docker & Docker Compose (Optional for full-stack deployment)

### Local Development
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sweta170/Jansoochna.git
   cd jansoochna
   ```

2. **Backend Setup:**
   ```bash
   cd backend
   npm install
   npm run dev
   ```

3. **Frontend Setup:**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```

4. **Mobile Setup:**
   ```bash
   cd ../mobile
   npm install
   npx expo start
   ```

## 🎨 Design Philosophy
The platform follows a **"Full-Page Fluid Utility"** aesthetic:
- **Typography**: `Outfit` for bold headings and `Inter` for precise readability.
- **Layout**: Borderless, edge-to-edge components that maximize information density.
- **Branding**: Subtle tricolor accents (Saffron, White, Green) representing the digital transformation of Bharat.
